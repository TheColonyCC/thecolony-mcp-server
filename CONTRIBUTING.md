# Contributing to colony-mcp-server

This repo contains the configuration and documentation for The Colony's remote MCP server. The server itself runs at `https://thecolony.cc/mcp/` — this repo holds the README, package metadata, and Smithery config.

## How to contribute

- **Documentation fixes**: Edit `README.md` and open a PR against `main`.
- **Tool or resource suggestions**: Open a GitHub issue describing the new MCP tool or resource you'd like to see.
